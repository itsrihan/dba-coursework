# NorthStar Databases and Analytics Coursework

## Project Overview

This project analyses the operational and analytical challenges affecting NorthStar Urban Mobility and Logistics using SQL within R, R analytics and visualisation, Python data processing, MongoDB Atlas, and query optimisation techniques.

The coursework investigates operational inefficiencies, customer complaints, delivery delays, service failures, route override behaviour, operational incidents, and fragmented data-management systems within a large-scale urban mobility and logistics environment.

The analysis demonstrates how relational and NoSQL analytical approaches can be combined to support operational monitoring, scalable analytics, and data-driven decision-making.
